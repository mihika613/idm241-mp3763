francesca's quickview - alpha details

1. URL to Original Interface:

Francesca's Charm Bar

2. Triggers:

"Add to Cart" button on the bottom right corner of the quickview modal in a product card

  • Hover on
  • Hover off
  • Click

3. Rules:

"Add to Cart" button

  • Hover on
    • Cursor changes to finger
    • Background of button changes color to black, text changes to white
  • Hover off
    • Cursor changes to arrow
    • Background of button changes color to white, text changes to black
  • Click
    • Text changes to “Adding...” with a spinning open circle indicating a loading sign
    • After a second or two, the text changes to “Added!” with a checkmark icon next to it
    • Adds item to the user's cart in the database
