francesca's quickview - alpha build

1. Triggers:

"Add to Cart" button on the bottom right corner of the quickview modal in a product card

  • Hover on
  • Hover off
  • Click

2. Rules:

"Add to Cart" button

  • Hover on
    • Cursor changes to finger
    • Background of button changes color to black, text changes to white
  • Hover off
    • Cursor changes to arrow
    • Background of button changes color to white, text changes to black
  • Click
    • Text changes to “Adding...” with a spinning open circle indicating a loading sign
    • After a second, the text changes to “Added!” with a checkmark icon next to it
    • Adds item to the user's cart in the database
    • Text changes back to "Add to Cart" after the checkmark icon with accompanying text "Added" have appeared on the button

3. Feedback:

"Add to Cart" button

  • Hover on
    • Cursor changes to finger
    • Background of button changes color to black, text changes to white
  • Hover off
    • Cursor changes to arrow
    • Background of button changes color to white, text changes to black
  • Click
    • Text changes to “Adding...” with a spinning open circle indicating a loading sign
    • After a second or two, the text changes to “Added!” with a checkmark icon next to it
    • Text changes back to "Add to Cart" after the checkmark icon with accompanying text "Added" have appeared on the button

4. Loops & Modes:

LOOPS:

"Add to Cart" button

  • Click
    • Circle icon next to "Adding..." continues to spin until the text changes to "Added!" with a checkmark icon next to it and the item is officially added to the cart in the database

MODES:

None
